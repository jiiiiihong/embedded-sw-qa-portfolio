- STM32F103RB 기반 임베디드 시스템 구현 경험
- GPIO, Timer, Interrupt, ADC, DMA, USART, NVIC 설정 경험

- ADC/DMA 기반 온도 센서 데이터 처리
- Timer별 역할 분리와 interrupt 기반 상태 제어 경험
- 핀 remap, 입력 bouncing, 하드웨어 결함 상황에서의 우회 구현 경험

- Isaac Sim 기반 2대 로봇 물류 분류 시뮬레이션 구현

## Vision & Validation
- OpenCV 기반 ROI 선택, threshold, contour, local peak 검출 경험

- 실제 동작 결과를 이미지/로그/JSON으로 남기는 검증 구조 경험

## Debugging & QA Mindset
- 기능 구현 후 실제 동작 결과를 확인하고 원인을 추적하는 방식으로 프로젝트 수행
- 센서 입력, 제어 로직, 하드웨어 설정, 시뮬레이션 조건을 분리하여 문제 분석
- 정상 시나리오뿐 아니라 timeout, fallback, 예외 전환 조건을 고려
- 재현 가능한 로그와 디버그 이미지를 남겨 검증 가능성을 높임
