# Braille Robot Validation

## 프로젝트 개요
로봇팔을 활용하여 맞춤형 점자를 제작하고, 제작된 점자를 비전 기반으로 검증하는 프로젝트를 수행했습니다.

## 사용 기술
- Robot Arm
- Python
- OpenCV
- Camera
- Image Processing
- Braille Pattern Validation

## 주요 구현 기능
- 점자 제작 흐름 구성
- 카메라 기반 점자 영역 인식
- ROI 기반 점형 검출
- 점형 판단을 통한 제작 결과 검증
- 한글 점자 표현 한계로 인한 다중 해석 가능성 고려

## 문제 해결
점자 표현 범위의 한계 때문에 동일한 점형에서 다양한 한글 해석군이 발생할 수 있었습니다.  
따라서 문자 해석 자체보다 제작된 점형이 목표 패턴과 일치하는지를 판단하는 방향으로 검증 기준을 조정했습니다.

## 배운 점
검증 시스템에서는 결과를 무리하게 해석하기보다, 판정 기준을 명확히 정의하고 실제 검증 가능한 단위로 문제를 재구성하는 것이 중요하다는 점을 배웠습니다.
