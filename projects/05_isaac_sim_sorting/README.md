# Isaac Sim Sorting Simulation

## 프로젝트 개요
NVIDIA Isaac Sim 환경에서 QR 코드가 부착된 박스를 인식하고, 두 대의 로봇이 출고일별로 박스를 분류하는 물류 자동화 시뮬레이션을 구현했습니다.

## 사용 기술
- NVIDIA Isaac Sim
- Python
- Robot Simulation
- QR Recognition
- Conveyor
- Multi-Robot Control

## 주요 구현 기능
- 컨베이어 기반 박스 유입 시나리오 구성
- QR 코드 기반 박스 정보 인식
- 출고일 기준 분류 로직 구현
- 단일 로봇 동작 검증
- 두 대의 로봇 기반 분류 동작 통합
- finalfac 창고 환경 기반 시뮬레이션 구성

## 개발 과정
초기에는 단일 로봇 기반 분류 동작 검증에서 시작했습니다.  
이후 QR 인식, 박스 방향, 컨베이어 흐름, 로봇 동작을 단계별로 통합하여 최종적으로 두 대의 로봇이 협동 분류하는 시뮬레이션을 구현했습니다.

## 문제 해결
- 로봇 내장 카메라 방식 폐기 후 월드 고정 카메라 방식으로 전환
- QR 코드가 보이는 면을 기준으로 박스 방향 설정
- 로봇 동작, QR 인식, 박스 생성 흐름을 통합 파일에서 관리
- 단계별 테스트를 통해 최종 시나리오 안정화

## 배운 점
시뮬레이션 환경에서도 실제 시스템과 마찬가지로 센서 위치, 객체 방향, 제어 타이밍, 로봇 동작 검증이 모두 맞물려야 한다는 점을 경험했습니다.
