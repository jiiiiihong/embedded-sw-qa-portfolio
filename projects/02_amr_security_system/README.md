# AMR Security System

## 프로젝트 개요
AMR을 활용하여 박물관 환경에서 도난 상황을 감지하고 추적하는 시스템을 구현했습니다.

## 사용 기술
- AMR
- RGB Camera
- Depth Camera
- LiDAR
- Python
- ROS
- Object Detection
- Path Planning
- Tracking Algorithm

## 주요 구현 기능
- 센서 기반 객체 탐지
- 도난 상황 발생 시 추적 모드 전환
- RGB/Depth/LiDAR 센서 정보를 활용한 경로 계획
- 두 대의 터틀봇이 동시에 추적하는 알고리즘 구현
- 교대 순찰 및 추적 신호 기반 동작 흐름 구성

## 문제 해결
센서 입력과 로봇 동작이 실시간으로 연결되어야 했기 때문에, 객체 탐지 결과와 로봇 이동 명령 사이의 흐름을 단계적으로 분리하여 검증했습니다.

## 배운 점
로봇 시스템에서는 단일 알고리즘 성능보다 센서 입력, 판단 로직, 이동 제어가 안정적으로 연결되는 구조가 중요하다는 점을 배웠습니다.
