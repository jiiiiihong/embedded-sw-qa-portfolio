# Screw Defect Detection System

## 프로젝트 개요
로봇팔과 RealSense 카메라를 활용하여 나사 체결 상태를 인식하고, 불량 여부를 판단한 뒤 조치하는 시스템을 구현했습니다.

## 사용 기술
- Robot Arm
- Intel RealSense Camera
- Python
- OpenCV
- Image Processing
- Defect Detection

## 주요 구현 기능
- RealSense 카메라 기반 체결 상태 인식
- 나사 체결 불량 판정
- 로봇팔 동작과 검사 결과 연동
- 불량 감지 후 조치 흐름 구성

## 문제 해결
카메라 입력 결과와 실제 체결 상태 사이의 차이를 줄이기 위해 인식 영역과 판정 기준을 조정했습니다.  
검사 결과가 로봇 동작으로 이어지는 흐름을 단계적으로 검증하며 시스템 안정성을 높였습니다.

## 배운 점
제조 검사 시스템에서는 단순 인식 정확도뿐 아니라, 검사 결과가 실제 조치 동작으로 이어지는 전체 흐름의 안정성이 중요하다는 점을 배웠습니다.
